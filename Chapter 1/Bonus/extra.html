CSS Properties

Color Property

color configures the text color.

This is a blue text.
- color: {blue or chosen color}


Background-Color Property

background-color configures the background color of the element.

This is a text with a green background.
- background-color: {green or chosen color}


Font-Family Property

font-family configures the type of font of the text.

This is a text with a Times New Roman font.
- font-family: {chosen font}


Font-Size Property

font-size configures the size of the text.

This is a text with a 20px font size.
- font-size: {chosen size}


Font-Style Property

font-style configures the style of the text (i.e normal, italic, oblique).

This is a text with an oblique font style.
- font-style: {oblique or chosen style}


Font-Weight Property

font-weight configures how thin or thick the lines of the text. This amount starts usually from 100 all the way to 900 and above.

This is a text with a font weight of 700.
- font-weight: {700 or chosen weight}


Line-Height Property

line-height configures the distance between one sentence and another vertically.

This is a text with a line height of 10px.

This is a text with a line height of 10px.

This is a text with a line height of 10px.

- line-height: {10px or chosen height}


Margin Property

margin configures the distance between elements.

This is a text with a margin of 50px.
- margin: {50px or chosen distance}


Text-Align Property

text-align configures the alignment of the text.

This is a text with a text alignment of right.
- text-align: {right or chosen alignment}


Text-Decoration Property

text-decoration configures the decorations added to the text and is shorthand property for:

  • text-decoration-line
  • text-decoration-color
  • text-decoration-style
  • text-decoration-thickness

This is a text with a text decoration of a lot.
- text-decoration: {chosen decoration}


Width Property

width configures the width of an element.

This is a text with a width of 50px. I'm gonna put some more text here so you can see.
- width {50px or chosen width}
